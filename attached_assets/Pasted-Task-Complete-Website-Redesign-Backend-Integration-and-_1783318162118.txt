Task: Complete Website Redesign, Backend Integration, and Automation

Mujhe apni website me jald se jald niche diye gaye sabhi advanced badlav (changes) karne hain. Har ek instruction ko dhyan se follow karein aur clean, fast-loading, aur production-ready code generate karein:

1. Ultra-Simple Layout & Zero Images (With Smooth Animations):
- Website ka layout bilkul simple, clean, lightweight aur minimal rakhna hai. Saare faltu elements hata dein.
- Website par koi bhi real ya AI-generated image use NA karein (0 images).
- Images ki jagah pure CSS/SVG-based smooth, clean aur modern animations (subtle fade-ins, hover effects, micro-interactions) ka use karein jo website ko super-fast rakhein.

2. App Features & Firebase Backend Configuration:
- Website ke backend me Firebase configuration set karein kyunki kuch app features ko Firebase se hi control kiya jata hai.
- Dhayan rahe, kisi bhi app ke andar koi direct API nahi hai. Saari APIs website ke backend par hi hosted aur integrated hain, isliye website backend ko central hub ki tarah structure karein.

3. Advanced Blog Section with Auto-Posting (No API):
- Blog section me categories aur ek functional Search Button hona chahiye.
- Main categories: Technology, News, Science, aur anya relevant topics.
- Manual article post karne ka time nahi hai, isliye backend me ek Web Scraping Technique (jaise Puppeteer ya Cheerio) ka use karein jo bina kisi paid API ke, har roj trusted sources se relevant articles dhundh kar website par automatically aur safely re-publish (publish) kar sake.

4. Contact Form via Cloudflare Email Routing:
- Contact form ke liye alag se koi heavy email service provider use nahi karna hai.
- Kyunki main pehle se hi Cloudflare Email Service use kar raha hoon, isliye contact form ke submission ko direct Cloudflare email routing ko hit karne ke liye configure karein, taki saare message seedhe mere email par forward ho jayein.

5. App Store Fallback, Smart Device Routing & Prominent Download Button:
- Website se Play Store ke saare links aur UI elements ko poori tarah se hata (remove) dein.
- Ek advanced Smart Routing Technique ka use karein (App Store Magic Link Button). Jab user is par click kare, toh script user ke device detection ke hisab se automatic routing execute kare.
- Agar user Apple/iOS par hai, toh yeh seedhe App Store name search ya direct identifier par redirect kare. 
- Agar user Android device par hai, toh yeh device ke andar maujud sabhi alternative native OEM app stores (jaise Samsung Galaxy Store, Xiaomi GetApps, Oppo App Market, Vivo App Store, Huawei AppGallery, Amazon Appstore) par proprietary internal URI schemes ke zariye package name ka automatic search trigger kare. Device me jo bhi app store available hoga, yeh bina kisi visual interruption ke seedhe wahan app ko download ke liye open kar dega (chahe wahan kitne bhi actors ya extra search results available hon, package name matching se exact app target ho).
- Ek prominent "Download APK / App" button ko user ke samne seedhe full screen par prominent tarike se display karein, aur agar koi native app store trigger nahi hota hai, toh yeh automatically hosted directory se fallback APK download shuru kar de.

6. Company & Owner Details Update:
- Website description text: "Nexus Wave Technologies is an innovative tech company dedicated to building accessible, high-performance digital solutions and mobile applications that empower users of all abilities."
- About Us page me company description ke sath-sath mera poora naam "Kuldeep Kumar Yadav" saaf-saaf include karein.

7. Update Privacy Policy & Terms and Conditions (For Both Apps):
Play Store ke compliance ke hisab se dono apps ke liye Privacy Policy aur Terms & Conditions ka content generate karein, jisme niche di gayi permissions ko saaf-saaf define aur justify kiya gaya ho:
- Location Permission: Emergency Guardian feature ko functional banane aur user ki safety ke liye.
- Media / File Manager Permission: App ke andar files aur media ko access/save karne ke liye.
